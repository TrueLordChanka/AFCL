# AFCL
The new and improved version of "FTW Scripts"

This liberary adds a host of scripts used in my personal, and other users H3 Mods.

AutoBolt -- Used to create the PTRD rifle <br>
AutoBoltHandle -- Used for the handle of the PTRD because I did not understand Mono-mod at the time<br>
bandgerSwitchInterp -- An attempt to make the switch on bangers interpolate for some reason, not used, unsure if it works<br>
betterOverheat -- A more advanced and open version of the overheat script which lets the user do alot more with their energy guns<br>
BreakableArmour -- Allows colliders to change their pmat, allowing for armour to "break" over time<br>
ShatterableArmour -- Allows a collections of colliders to act as one piece, breaking all when the damage threshold is reached<br>
ShatterableArmourPiece -- A component used on the individual colliders.<br>
EnableOnSpecificUser -- Reaches out to Steam servers to either enable or disable an object when a specific user is playing<br>
EnergySword -- Code used for the energy sword weapon, unreleased<br>
FireSubprojAtDistance -- Fires a projectile when a parent projectile travels a certain distance<br>
FireSubprojAtTime -- Fires a projectile when a parent projectile exists for an amount of time CURRENTLY NOT WORKING<br>
ForeGripMagRelease -- The code that allows the foregrip of the Vector to perform the funcitons it does. Thanks to CityRobo for making this all work<br>
MultiEnableLaserPointer -- Allows two objects to turn on whith the base game laser code. used in IR lasers<br>
NightVisionPostEffect -- Written by devyndamonster, used for Nightvision<br>
PostEffectScript --Written by devyndamonster, Used for Nightvision<br>
PrefabLoaderReadme -- a dumb readme i made to demonstrate usage of the new prefab loader in the template project<br>
proxGrenade -- A proximity Grenade used for the Grzmot mine. Absolute spagetii code.<br>
proxGrenadeRing -- used for above, didnt know hookgen<br>
StickyObjct -- All commented out, does nothing, when did i make this?<br>
thermiteFire -- Creates the fire effect used in my thermite grenade<br>
ToggleBetweenPoints -- toggels an object while between two points<br>
